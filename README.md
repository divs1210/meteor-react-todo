# meteor-react-todo

Full-stack reactive webdev starter template.

## Includes:

- TypeScript
- user authentication
- server side rendering
- client side routing

## Install

Meteor requires `node 14`, which can be installed via [nvm](https://github.com/nvm-sh/nvm):

```
$ nvm install 14
$ nvm use 14
$ meteor npm install
```

## Run

```
$ meteor
```
